伪分布模式

配置xml文件

文件均在/usr/local/hadoop/etc/hadoop/

在<configuration></configuration>之间增加内容

core-site.xml包含了Hadoop启动时的配置信息。

yarn-site.xml包含了MapReduce启动时的配置信息。

mapred-site.xml用于指定MapReduce使用的框架

hdfs-site.xml用来配置集群中每台主机都可用，指定主机上作为namenode和datanode的目录。

可以在别的路径下创建name和data文件夹，名称也可以与不同，但是需要和hdfs-site.xml中的配置一致。
