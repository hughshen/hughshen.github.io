搭建Hexo博客，并部署到Github上

2015-03-28

安装Hexo

ubuntu下只有几条命令

在安装nvm的时候，连接不成功，直接到Github下载nvm的文件，然后运行install.sh。

搭建本地博客

在浏览器输入localhost:4000就能看到新建的博客了。具体

部署到Github

在Github上新建一个仓库，仓库名称为账户名称.github.com，必须是这个！！

然后在博客目录下建立仓库，可能需要配置git信息

接下来编辑配置文件_config.yml

编辑保存之后就可以进行推送

另外，在推送的时候 出现ERROR Deployer not found: github，具体看#1040

解决：把github改为git之后，还需要在终端npm install hexo-deployer-git --save，还有注意空格！

Hexo基本上就搭建好了，有空再深入了解。
