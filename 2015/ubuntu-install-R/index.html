安装RStudio

RStudio直接下载deb包用dpkg安装就行，对于RStudio Server 具体

安装完之后，可以使用，但是有两个小问题。

1.编辑Source文件不能保存，弹出no such file or dir错误。

解决：找到RStudio的启动文件，然后修改其用户权限chmod u+s ***（可执行文件名）；或者在建立workspace文件夹的时候，直接建立，不要使用sudo。

2.安装包失败，显示installation of package 'XML' had non-zero exit status或者package 'rvest' is not available (for R version 3.1.3)

解决：原因是这些包依赖了特别的库 参考

  • Cairo：sudo apt-get install libcairo2-dev libxt-dev
  • cairoDevice：sudo apt-get install r-cran-cairoDevice
  • rgl：sudo apt-get install r-cran-rgl
  • fftw：sudo apt-get install libfftw3-dev
  • rcdd：sudo apt-get install libgmp3-dev
  • rgdal：sudo apt-get install libgdal-dev libproj-dev
  • XML：sudo apt-get install libxml++2.6-dev
  • rimages：sudo apt-get install fftw-dev fftw2
  • RCurl：sudo apt-get install libcurl4-gnutls-dev，不行的话再安装libcurl4-openssl-dev

最后install.packages("xxx")即可
